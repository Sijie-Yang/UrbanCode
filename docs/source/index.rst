Welcome to UrbanCode's documentation!
================================

UrbanCode is a Python package for urban analysis and computer vision tasks. It provides tools for:

* Street View Image (SVI) analysis
* Semantic segmentation
* Object detection
* Scene recognition
* Network analysis
* Urban perception
